# Virtual Painter – Hand Gesture Detection 🎨✋

This project is a **Computer Vision–based Virtual Painter** that allows users to draw on the screen using **hand gestures** detected via a webcam.  
It uses real-time hand landmark detection to track finger movements and draw digitally.

## 🚀 Features
- Hand tracking using OpenCV & MediaPipe
- Draw on screen using finger gestures
- Real-time webcam interaction
- Multiple color selection

- ## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
